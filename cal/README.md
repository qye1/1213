# Калькулятор (Практическая работа №2)

## Описание
Веб-приложение «Калькулятор», разработанное с использованием системы контроля версий Git.

## Функционал
- Сложение
- Вычитание
- Умножение
- Деление
- Остаток от деления
- Sin / Cos
- Возведение в степень
- Квадратный корень
- Округление (floor, ceil)
- Работа с памятью (M+, MC)

## Технологии
- HTML
- CSS
- JavaScript
- Git / GitHub

## Запуск проекта
Открыть файл `index.html` в браузере.
